# 🌍 Traveloop

Traveloop is a modern AI-powered travel planning web application developed for hackathon projects using Flask, HTML, CSS, JavaScript, and SQLite.

The platform helps users organize trips, manage itineraries, calculate budgets, track expenses, create packing checklists, store travel notes, and share travel plans with others.

---

# ✨ Features

## 🔐 User Authentication
- Secure Signup & Login System
- Password Hashing
- Session Management using Flask-Login

## 🧳 Trip Management
- Create Trips
- View Upcoming Trips
- Manage Multiple Destinations

## 🗺️ Smart Itinerary Builder
- Add Multiple Stops
- Organize Travel Schedule
- Manage Activities for Each Stop

## 💰 Budget Planner
- Expense Tracking
- Cost Breakdown
- Trip Budget Overview

## ✅ Packing Checklist
- Add Packing Items
- Mark Items as Packed
- Category-wise Organization

## 📝 Travel Notes
- Save Important Notes
- Personal Trip Documentation

## 📦 Booking Management
- Store Hotel/Flight/Train Details
- Save Booking Confirmation Numbers

## 👥 Shared Itinerary
- Public Trip Sharing
- Easy Collaboration

## ⭐ Reviews & Ratings
- Add Travel Reviews
- Rate Experiences

## 🛠️ Admin Dashboard
- User Overview
- Trip Analytics
- Platform Monitoring
